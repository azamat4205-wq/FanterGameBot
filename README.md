# 🎮 FanterGameBot

Игровой Telegram-бот на aiogram 3.

## Игры

- 🪙 Орёл и решка
- ❌⭕ Крестики-нолики
- 😀 Найди пару
- 🔫 Русская рулетка
- 🔴🟡 4 в ряд
- ⚡ Кто быстрее нажмёт кнопку

## Возможности

- 👥 Игра с другом
- 🤖 Игра с ботом
- 🔗 Приглашение по ссылке
- 🏆 Рейтинг
- 👤 Профиль
- 🔄 Игры обновляются в одном сообщении
